1. Image also have two styles, just like links, and both of them render exact same way.

Inline Image Link:

2. Images are preface with an exclamation point (!)
3. The first image style is called inline image link.
For example,
![Benjamin Bannekat](https://octodex.github.com/images/bannekat.png).

Reference Image Link:

You'll precede the Markdown with an exclamation point, then provide two brackets for the alt text, and then two more for the image tag, like this: ![The founding father][Father] At the bottom of your Markdown page, you'll define an image for the tag, like this: [Father]: http://octodex.github.com/images/founding-father.jpg.
